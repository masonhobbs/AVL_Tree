AVL Tree

Self-balancing binary tree that uses node rotations to maintain the balance property of AVL trees.
Test cases are in the form of text files with commands to either insert a node, delete a node, or print
the tree.


Compile:

  $ g++ -std=c++0x avl.cpp
  
Run:
  $ ./a.out testcases.txt
